Калькулятор коэффициентов Demand/SWAT
======================================

Веб‑приложение для автоматического расчёта корректировочных коэффициентов между историческими данными `DEMAND` и прогнозами `SWAT`.  
Приложение работает как **калькулятор в браузере**: вы загружаете файлы, получаете результат и можете скачать отчёт.

Сайт доступен по ссылке:  
https://pushkinhub.github.io/coefficients-calculator/

## Безопасность и конфиденциальность

- Все вычисления выполняются **целиком в вашем браузере**.
- Загруженные CSV‑файлы **не отправляются на сервер**, не сохраняются во внешних базах данных и не передаются третьим лицам.
- Страница развёрнута на **GitHub Pages** как статический сайт, без собственного бэкенда и API – файлы используются только локально в рамках вашей браузерной сессии.
- Приложение не использует дополнительные сетевые протоколы для передачи содержимого файлов – они обрабатываются так же, как данные в обычном локальном калькуляторе.

Таким образом, приложение можно рассматривать как безопасный локальный инструмент для расчёта коэффициентов, который просто открывается в окне браузера.

## Правила расчёта коэффициентов

- **`coefficient_raw`** – отношение суммы `Demand` за выбранный период к сумме `SWAT` за тот же период.
- **`coefficient_adjusted`** – скорректированное значение коэффициента по правилам:
  - если получилось `0` или ошибка → берём `1.00`;
  - если значение в диапазоне от `0.96` до `1.04` → приводим к `1.00`;
  - если значение меньше `0.8` → устанавливаем `0.80`;
  - если значение больше `1.5` → устанавливаем `1.50`.

## Формулы метрик

Рассчитанные метрики: `Coefficient`, `Difference`, `Bias %`, `OSA %`, `Writeoffs %`.

- **Bias %**: (prediction_final - demand) / demand * 100
- **Difference**: prediction_final - demand
